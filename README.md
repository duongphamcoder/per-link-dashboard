  - You can install via [nvm](https://github.com/creationix/nvm)
- [Pnpm](https://pnpm.io/) (Optional) &gt;=8.6.7

## Getting Started

Step by step to run this app in your local

### Install packages

At your directory root, run:

```bash
pnpm install
```

### Run server

```bash
pnpm run dev
```

### Build

```bash
pnpm run build
```
